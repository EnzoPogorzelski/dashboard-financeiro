Importar planilha

Use Excel .xlsx ou CSV com as colunas: Mês, Tipo, Bloco, Categoria, Item e Valor.

Dica: mantenha sua planilha de lançamentos no computador. Quando quiser atualizar o site, importe o arquivo novamente.

Backup e exportação

Baixe uma cópia dos dados salvos no navegador para guardar ou editar em Excel.

Os dados do dashboard ficam no navegador em que você usou o site. Em outro computador/celular, importe o backup ou a planilha novamente.
Não suba sua planilha financeira preenchida no GitHub público. Suba apenas os arquivos do site.

Melhor jeito de usar

Esta versão foi pensada para não precisar editar código no GitHub. Você publica o site uma vez e depois atualiza os dados pelo próprio dashboard.

Fluxo recomendado:
1. Publique este site no GitHub Pages.
2. Abra o site e digite a senha.
3. Importe a planilha de lançamentos.
4. Edite dados na aba Lançamentos ou reimporte uma planilha atualizada.
